# 用語集

対象アプリ: `zend-framework-1-crud-master`

## アプリ固有

| 用語 | 意味 |
| --- | --- |
| Ticket | このアプリの中心データ。チケット1件分を表す |
| tickets | チケットを保存するDBテーブル |
| Topic | ファイル名や変数名に残る表現。実質的にはTicketを指している箇所が多い |
| CVS export | READMEや画面にある表記。意図はCSV exportと思われる |

## Zend Framework 1関連

| 用語 | 意味 |
| --- | --- |
| Controller | URLごとの処理を受けるクラス |
| Action | Controller内の具体的な処理メソッド |
| View | HTMLを組み立てるテンプレート |
| Layout | 全画面共通の外枠テンプレート |
| Bootstrap | アプリ起動時の初期化処理 |
| Route | URLとController/Actionの対応 |
| Zend_Form | フォーム定義・検証を行うZF1コンポーネント |
| Zend_Db_Table_Abstract | DBテーブル操作用のZF1基底クラス |
| Zend_Paginator | ページング処理 |
| Action Helper | Controllerから呼び出す補助処理 |
| View Helper | Viewから呼び出す補助処理 |
| FlashMessenger | リダイレクト後にメッセージを表示するための仕組み |

## このアプリでのクラス

| クラス | 役割 |
| --- | --- |
| `TicketController` | [チケット一覧](../画面再現/01_チケット一覧.png)、登録、編集、削除、CSV出力を担当 |
| `Application_Model_Ticket` | チケット1件分のデータ入れ物 |
| `Application_Model_TicketMapper` | ControllerとDB操作の橋渡し |
| `Application_Model_DbTable_TicketDbTable` | `tickets` テーブルに対応 |
| `Application_Form_TopicBootstrapForm` | 登録・編集フォーム |
| `Zend_Controller_Action_Helper_Csv` | CSV出力Helper |
| `Zend_View_Helper_DisplayDate` | 日付表示Helper |
