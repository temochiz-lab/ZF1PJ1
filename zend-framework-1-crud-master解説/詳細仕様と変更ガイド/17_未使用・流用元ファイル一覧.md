# 未使用・流用元ファイル一覧

対象アプリ: `zend-framework-1-crud-master`

## 概要

このアプリには、ZF1プロジェクト生成時の名残や、別サンプルから流用されたと思われるファイルがある。

完全に未使用と断定するには実行時確認が必要だが、主要なチケットCRUDからは直接使われていない可能性が高い。

## 未使用・低使用の可能性があるファイル

| ファイル | 理由 | 確認ポイント |
| --- | --- | --- |
| [`application/controllers/IndexController.php`](../各ファイル解説/application/controllers/IndexController.php_解説.md) | `/` は `TicketController#indexAction` にルーティングされる | [`routes.php`](../各ファイル解説/application/configs/routes.php_解説.md) |
| [`application/views/scripts/index/index.phtml`](../各ファイル解説/application/views/scripts/index/index.phtml_解説.md) | `IndexController` が主要導線で使われない | 直接URLで使われる可能性は確認要 |
| [`application/forms/----TopicForm.php`](../各ファイル解説/application/forms/----TopicForm.php_解説.md) | 現在のControllerは `TopicBootstrapForm` を使用 | [`TicketController.php`](../各ファイル解説/application/controllers/TicketController.php_解説.md) |
| [`application/views/scripts/ticket/delete.phtml`](../各ファイル解説/application/views/scripts/ticket/delete.phtml_解説.md) | 削除後すぐリダイレクトするため表示されにくい | `deleteAction()` |
| [`application/helpers/Multiples.php`](../各ファイル解説/application/helpers/Multiples.php_解説.md) | コメントアウトされたテスト用途に近い | `TicketController#cvsexportAction` |
| [`application/views/helpers/DisplayDateLab.php`](../各ファイル解説/application/views/helpers/DisplayDateLab.php_解説.md) | [`DisplayDate.php`](../各ファイル解説/application/views/helpers/DisplayDate.php_解説.md) と役割が近い | View内の呼び出し確認 |
| [`application/views/helpers/DisplayAddress.php`](../各ファイル解説/application/views/helpers/DisplayAddress.php_解説.md) | チケットに住所項目がない | View内の呼び出し確認 |
| [`application/views/helpers/LoggedInUser.php`](../各ファイル解説/application/views/helpers/LoggedInUser.php_解説.md) | 認証Controller/Modelが不足 | Layout/View内の呼び出し確認 |

| `users` テーブル | 主要なチケットCRUDで使われていない | 認証機能なし |
| `other_example/*` | 参考資料または別サンプル | 実行時には不要の可能性 |

## 命名の名残

| 箇所 | 内容 |
| --- | --- |
| [`routes.php`](../各ファイル解説/application/configs/routes.php_解説.md) | ルート名に `labs/create` などが残っている |
| README | `cvs export` と書かれているが、意図は `csv export` と思われる |
| `TicketMapper` コメント | `Blog` や `lab topics` の名残がある |
| [`Ticket.php`](../各ファイル解説/application/models/Ticket.php_解説.md) 例外文言 | `Invalid guestbook property` の名残がある |

## 削除前の確認手順

```text
1. ファイル名・クラス名・メソッド名で検索する
2. routes.php と layout.phtml を確認する
3. View Helper / Action Helper の呼び出し形式を確認する
4. 実行時にアクセスされる可能性があるURLを確認する
5. 削除ではなく、まず未使用候補として記録する
```

## 注意

ZF1では、クラス名やディレクトリ構造から自動的に読み込まれることがある。

単純な文字列検索で出ないからといって、即削除してよいとは限らない。
