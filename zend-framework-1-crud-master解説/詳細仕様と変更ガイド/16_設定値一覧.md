# 設定値一覧

対象アプリ: `zend-framework-1-crud-master`

## application.ini

ファイル:

- [`application/configs/application.ini`](../各ファイル解説/application/configs/application.ini_解説.md)

| 設定 | 値 | 内容 | 注意 |
| --- | --- | --- | --- |
| `phpSettings.display_startup_errors` | `1` | 起動時エラー表示 | 本番では無効化検討 |
| `phpSettings.display_errors` | `1` | PHPエラー表示 | 本番では無効化検討 |
| `includePaths.library` | `APPLICATION_PATH "/../library"` | ZF1 libraryパス | [`library.rar`](../各ファイル解説/library.rar_解説.md) の展開が必要と思われる |
| `bootstrap.path` | `APPLICATION_PATH "/Bootstrap.php"` | Bootstrapファイル | 起動時に使用 |
| `bootstrap.class` | `Bootstrap` | Bootstrapクラス名 |  |
| `appnamespace` | `Application` | アプリ名前空間 | Model/Form等のクラス名に影響 |
| `resources.frontController.controllerDirectory` | `APPLICATION_PATH "/controllers"` | Controllerディレクトリ |  |
| `resources.frontController.params.displayExceptions` | `1` | 例外表示 | 本番では無効化検討 |
| `resources.layout.layoutPath` | `APPLICATION_PATH "/layouts/scripts/"` | Layoutパス | [`layout.phtml`](../各ファイル解説/application/layouts/scripts/layout.phtml_解説.md) |
| `resources.view[]` | 空 | View初期化 |  |
| `resources.db.adapter` | `PDO_MySQL` | DBアダプタ | MySQL前提 |
| `resources.db.params.host` | `localhost` | DBホスト | 環境依存 |
| `resources.db.params.username` | `root` | DBユーザー | 本番では危険 |
| `resources.db.params.password` | 空 | DBパスワード | 本番では危険 |
| `resources.db.params.dbname` | `zf1app_db` | DB名 | SQLファイルと対応 |
| `resources.db.params.charset` | `utf8` | 文字コード |  |

| 定数 | 内容 |

## routes.php

| URL | Controller | Action |
| --- | --- | --- |
| `/` | `ticket` | `index` |
| `ticket/create` | `ticket` | `save` |
| `ticket/edit/:id` | `ticket` | `edit` |
| `ticket/delete/:id` | `ticket` | `delete` |
| `ticket/cvsexport` | `ticket` | `cvsexport` |

## 設定変更時の注意

- DB接続情報は平文で書かれている。
- 例外表示とPHPエラー表示が有効になっている。
- `UPLOAD_PATH` を変える場合、フォームとアップロード処理も確認する。
- ルート名に `labs/*` が残っているが、実URLは `ticket/*`。
